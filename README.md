Payback² Mod APK. Creator: Ash3s_<br><br>

## Details

## Mod Features 👻

- Adds 12 new maps and boat races to the game
- Includes 11 maps ported from Payback 1
- 6 maps are from the GBA version
- 5 maps are newer maps from the original website
- Adds a water map and replaces the Pug Racer with a boat for boat races
- Updated map names and strings
- New intro and music
- Changed package name
- No ads
- Infinite coins
- Fully independent from Google Play Games and the original game, allowing both versions to be installed side by side.

## Known Issues ⚡
Some surfaces, especially certain rooftops, may have unusual textures. Most textures were converted correctly, but a few are not fully compatible with Payback 2.

Also the fact that sometimes the render inGame, lacks a bit.
